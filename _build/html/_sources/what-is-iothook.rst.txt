################
What is IOTHOOK?
################

IoThook is a web service network project that transfers data between internet-connected objects (IoT, IIoT).
IoThook connects Arduino, Raspberry Pi, Android, iOS, Windows Phone, Website, Banana Pi, Orange Pi, Beaglebone,
ARM, Pic, Windows, Mac OS X, and Linux based systems. It supports HTTP and MQTT protocols. With IoThook, you can
send and receive data via HTTP and MQTT, and record data. IoThook is open source. <https://iothook.com/>
